# 易拼图

基于 Next.js 的在线拼图网站，自带多种照片拼图模板来帮助您完成设计，并可以让您轻松调整背景和效果。

## Installation

```bash
npm install
```

## Development

First, run the development server:

```bash
npm start
```

or

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
